# 🚀 Inventory Management System (Fullstack + Mobile)

This project is a complete **fullstack inventory management system** with:

* 🌐 Web application (React)
* 📱 Mobile application (Flutter)
* ⚙️ Backend API (Laravel)
* ☁️ Cloud deployment (Railway + Vercel)

---

## 🧱 Project Structure

### 🔹 1. Backend (Laravel API)

* Handles authentication (Laravel Sanctum)
* Provides REST API for products
* Connected to MySQL database
* Deployed on Railway

🔗 **Live API:**
https://inventory-api-production-8068.up.railway.app/api

📂 Repository:
`inventory-api`

---

### 🔹 2. Web App (React)

* Modern UI with Tailwind CSS
* Login & Register system
* Token-based authentication
* Full CRUD for products

🔗 **Live Web App:**

* https://nventory-web-five-sooty.vercel.app
* https://inventory-web-git-main-dutchjohn80-rgbs-projects.vercel.app
* https://inventory-1r1z2j148-dutchjohn80-rgbs-projects.vercel.app

📂 Repository:
`inventory-web`

---

### 🔹 3. Mobile App (Flutter)

* Login & Register
* Token storage (SharedPreferences)
* Auto-login system
* View and add products
* Connected to same Laravel API

📂 Repository:
`inventory-mobile`

---

## 🔐 Authentication

* Implemented using **Laravel Sanctum**

* Shared across:

  * React (Web)
  * Flutter (Mobile)
